# Toby Shevlane

## Profile
Researcher at Google DeepMind. Co-lead of the dangerous capability evaluations programme. Lead author of Shevlane et al. (2023) "Model Evaluation for Extreme Risks," which established the framework that Phuong et al. (2024) builds upon.

## Key Contributions
- [[sources/shevlane-et-al-2023]] — lead author; foundational framework for model evaluation for extreme risks; proposed nine dangerous capability categories and governance integration
- [[sources/phuong-et-al-2024]] — co-led dangerous capability evaluation programme; led forecasting component
